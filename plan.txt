# taplink.my — Tam UI/UX Spesifikasyonu
> Bu doküman AI'ya (Claude Code, Cursor, vs.) sayfa sayfa kod yazdırmak için hazırlanmıştır.
> Her sayfayı sırayla ver. Bir sayfa bitmeden diğerine geçme.

---

## Genel Kurallar

- **Framework:** Django 5.x + Django Templates + TailwindCSS v3 + Alpine.js v3
- **Renk modu:** Dark mode varsayılan (sistem geneli)
- **Dil:** İngilizce — i18n altyapısı hazır olsun, çeviri dosyaları boş kalabilir
- **Responsive:** Mobil öncelikli — profil sayfası max 640px ortalanmış, dashboard tam ekran
- **Font:** Google Fonts API entegrasyonu — kullanıcı istediği fontu seçebilir
- **Auth:** Sadece Google OAuth (django-allauth) — şifre sistemi yok, SMTP yok
- **Subdomain:** `kullaniciadi.taplink.my` — Caddy wildcard SSL

### Sekme 1 — Links

**Üst kısım:**
- "Add link" butonu (sağ üst)
- Plan limiti göstergesi: "2/2 links used" (Free) veya "3/10 links" (Standard)
- Free kullanıcı limitte: Sarı uyarı banner + "Upgrade for more links"

**Link ekleme:**
- "Add link" tıklanınca listenin en altına yeni boş kart eklenir (inline)
- Kart otomatik açılır, başlık inputuna fokus gider

**Her link kartı:**
- Sol: Drag handle (≡ ikonu) — sürükle bırak sıralama
- Sağ üst: Yukarı/aşağı ok butonları (alternatif sıralama)
- Alanlar (inline, kaydetme otomatik — blur veya Enter'da):
  - İkon seçici (grid popup — tüm platformlar)
  - Başlık input
  - URL input
  - Özel renk seçici (sadece bu link için)
  - Görünüm toggle: "Icon + text" veya "Icon only"
  - Thumbnail: URL gir → otomatik çeker; başarısız olursa "Upload image" gösterir
  - Aktif/pasif toggle (sağ üst köşe)
- Sağ alt: Silme butonu (kırmızı, onay ister)

**İkon seçici popup:**
Grid layout, kategoriler:
- Social: TikTok, Instagram, YouTube, Facebook, X (Twitter), Pinterest, Snapchat, LinkedIn, Reddit, Twitch, Discord, Telegram, WhatsApp, WeChat, LINE, Viber, BeReal, Threads
- Shopping: Shopee, Lazada, Amazon, eBay, Etsy, WooCommerce
- Food & Delivery: GrabFood, Foodpanda, LINE MAN, Robinhood
- Travel: Grab, Bolt, Airbnb, Booking.com, Agoda, Klook
- Music: Spotify, Apple Music, SoundCloud, YouTube Music
- Other: Website, Email, Phone, PDF, Location, Custom (kullanıcı SVG yükler)

### Sekme 2 — Appearance

**Alt bölümler (accordion veya scroll):**

#### Profil
- Profil fotoğrafı yükle (drag & drop veya click, max 2MB, kırpma aracı)
- Fotoğraf şekli: Yuvarlak / Kare / Altıgen (buton grubu)
- Border rengi: Color picker
- Border kalınlığı: Slider (0–8px)
- Display name: Text input
- Bio: Textarea (max 160 karakter, sayaç gösterir)
- Konum: Text input (şehir, ülke)
- Verify rozeti: Toggle (sadece admin aktif edebilir — kullanıcı görebilir ama değiştiremez)

#### Arka Plan
- Tip seçimi (buton grubu): Düz Renk / Gradient / Fotoğraf / Video
- **Düz Renk:** Color picker
- **Gradient:** İki renk seçici + yön (yatay/dikey/köşegen)
- **Fotoğraf:** Yükle (max 5MB) + pozisyon ayarı (center/top/bottom)
- **Video:** YouTube/Vimeo URL gir veya dosya yükle (max 30MB, .mp4) — sadece Standard

#### Tema & Butonlar
- Buton stili (grid seçim):
  - Filled (dolu arka plan)
  - Outline (sadece kenarlık)
  - Soft (hafif arka plan)
  - Shadow (gölgeli)
  - Glass (cam/blur efekti)
  - Gradient (iki renkli gradient buton)
- Buton rengi: Color picker
- Buton yazı rengi: Color picker
- Buton border radius: Slider (0–50px, 0=köşeli, 50=tam yuvarlak)
- Buton hover efekti: Toggle (açıksa mouse üzerinde hafif parlama)

#### Yazı Tipi
- Google Fonts arama kutusu (live search)
- Popüler fontlar hızlı seçim için üstte gösterilir (Inter, Poppins, Kanit, Sarabun, Prompt)
- Yazı boyutu: Slider (küçük/orta/büyük)
- Yazı rengi: Color picker

#### Sosyal İkonlar
- Hangi ikonların profil sayfasında gösterileceğini seç
- İkon stili: Renkli / Beyaz / Siyah
- İkon boyutu: Küçük / Orta / Büyük

## SAYFA 6 — Herkese Açık Profil (`kullaniciadi.taplink.my`)

### Genel
- Mobil öncelikli, max 640px, ortalanmış
- Dark/Light: Kullanıcının seçtiği arka plana göre otomatik
- Hızlı yükleme — minimal JS

### İçerik sıralaması (yukarıdan aşağı):
1. Arka plan (renk/gradient/fotoğraf/video — kullanıcının seçimine göre)
2. Profil fotoğrafı (seçilen şekil + border)
3. Verify rozeti (eğer aktifse — mavi tik ikonu)
4. Display name
5. Konum (varsa — küçük pin ikonu ile)
6. Bio
7. Sosyal ikonlar (küçük, yatay sıra)
8. Linkler (aşağı doğru liste)
9. Watermark (Free'de: taplink.my logosu + "Create your page" CTA)

### Link davranışı
**Free:** Tıklanınca direkt yönlendir (yeni sekme)

**Standard:**
- Tıklanınca kısa animasyon (ripple efekti)
- Kısa splash screen (0.5sn — platform ikonu + "Opening..." yazısı)
- Sonra yönlendir
- Tıklanma analytics'e kaydedilir

### Link görünümü
- Kullanıcının seçimine göre: İkon + yazı veya sadece ikon
- Thumbnail varsa: Sol tarafta küçük görsel
- Buton stili, renk, border: Kullanıcının appearance ayarları

### SEO
- `<title>`: Display name + "| taplink.my"
- `<meta description>`: Bio
- Open Graph tags (Facebook/WhatsApp önizleme):
  - `og:title`: Display name
  - `og:description`: Bio
  - `og:image`: Profil fotoğrafı
  - `og:url`: Profil URL'i
- JSON-LD schema (Google arama):
  - Person schema: isim, bio, url, sameAs (sosyal linkler)

### Hata durumları
- Kullanıcı bulunamazsa: "This page doesn't exist" (taplink.my'e yönlendirme)
- Sayfa duraklıysa: "This page is currently unavailable"

## Genel UX Kuralları

### Toast Bildirimleri
Her kaydetme, silme, hata işleminde:
- Sağ alt köşede çıkan toast
- ✅ Yeşil: Başarı
- ❌ Kırmızı: Hata
- ⚠️ Sarı: Uyarı
- 3 saniye sonra otomatik kaybolur

### Yükleme Durumları
- Butonlar tıklanınca spinner gösterir, disabled olur
- Sayfalar arası geçişte skeleton loader

### Boş Durumlar (Empty States)
- Links sekmesi boşken: İllüstrasyon + "Add your first link" butonu
- Analytics boşken: "Share your page to start getting views"

### Mobil Dashboard
- Sidebar varsayılan kapalı, hamburger menü
- Preview varsayılan kapalı, "Preview" butonu ile açılır
- Tek panel görünümü

### Güvenlik
- Her view'da `request.user == profile.user` kontrolü
- CSRF token tüm POST/AJAX
- Dosya yükleme: avatar max 2MB, background max 5MB, video max 30MB
- Rate limiting: login endpoint'ine dakikada max 10 istek
- Subdomain olarak yasaklı kelimeler: admin, www, api, static, media, mail, ftp, blog, app, dashboard, login, register, support, help, about, contact, terms, privacy, taplink, billing, upgrade, redeem, qr
